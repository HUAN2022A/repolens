# RepoLens v0.1.0

RepoLens v0.1.0 is the first public release of a local-first context generator for AI coding agents.

## Highlights

- Generate compact Markdown context packs for Codex, Claude Code, Cursor, and generic coding agents.
- Scan local repositories or public GitHub repository URLs.
- Respect common ignore rules and `.gitignore` patterns.
- Rank task-relevant files and explain relevance reasons in `repo-map.json`.
- Choose all outputs, JSON-only output, or Markdown-only output.
- Run with no runtime dependencies on Node.js 18+.

## Install from source

```bash
git clone https://github.com/HUAN2022A/repolens.git
cd repolens
npm test
npm run smoke
node src/cli.js . --task "understand this repo" --for codex
```

## Validation

- `npm test`
- `npm run smoke`
- `npm pack --dry-run`
- GitHub Actions CI on Node.js 18 and 20

## Known limitations

- GitHub URL support relies on local `git clone`.
- `.gitignore` support covers common patterns, not the full gitignore specification.
- Code understanding is heuristic; AST/tree-sitter symbol graphs are planned for later releases.
