# RepoLens v0.2.0

RepoLens v0.2.0 adds a minimal dependency-free stdio MCP server, moving RepoLens from a CLI-only context generator toward agent context infrastructure.

## Highlights

- New `repolens-mcp` bin for MCP-style stdio tool calls.
- New `generate_context` tool to write Markdown and/or JSON context packs.
- New `repo_map` tool to return machine-readable context without writing Markdown files.
- Shared context generation layer between CLI and MCP server.
- MCP test coverage through Node's built-in test runner.
- Improved README, demo docs, benchmark notes, and packed CLI verification.

## Usage

```bash
node src/mcp.js
# or after installation
repolens-mcp
```

See [`docs/mcp.md`](../mcp.md) for JSON-RPC examples and tool schemas.

## Validation

- `npm test`
- `npm run smoke`
- `npm run pack:test`
- GitHub Actions CI on Node.js 18 and 20

## Notes

The MCP server is intentionally minimal and dependency-free. Future releases can add richer MCP compatibility, resources, and setup snippets for specific coding agents.
