# RepoLens v0.3.0

RepoLens v0.3.0 improves the MCP agent workflow with a lightweight task-routing tool and setup documentation.

## Highlights

- New `find_relevant_files` MCP tool.
- `find_relevant_files` returns a compact list of task-relevant files without writing context files or returning a full repo map.
- Added MCP setup examples for generic clients, Claude Desktop-style configs, and editor agents.
- Expanded MCP docs with JSON-RPC examples for all tools.

## MCP tools

- `generate_context`: write Markdown and/or JSON context packs.
- `repo_map`: return the full machine-readable repo map.
- `find_relevant_files`: return the top relevant files for a task.

## Validation

- `npm test`
- `npm run smoke`
- `npm run pack:test`
- GitHub Actions CI on Node.js 18 and 20

## Recommended agent flow

1. Call `find_relevant_files` with the user's task.
2. Read the top files.
3. Call `repo_map` if broader structure is needed.
4. Implement and verify using the repository's existing checks.
