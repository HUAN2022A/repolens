# RepoLens v0.6.0

RepoLens v0.6.0 prepares the project for npm publishing under the scoped package name `@huan2022a/repolens`.

## Highlights

- Package metadata now uses `@huan2022a/repolens` because the unscoped `repolens` package name is already occupied on npm.
- CLI command names remain unchanged: `repolens` and `repolens-mcp`.
- README and agent setup docs now include npm installation guidance.

## Install after npm publish

```bash
npm install -g @huan2022a/repolens
repolens . --task "understand this repo" --for codex
repolens-mcp
```

## Publish command

After logging into npm with an account that owns the `@huan2022a` scope:

```bash
npm login
npm publish --access public
```

## Validation

- `npm test`
- `npm run smoke`
- `npm run pack:test`
- `npm pack --dry-run`
- GitHub Actions CI on Node.js 18 and 20
