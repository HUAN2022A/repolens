# RepoLens v0.8.0

RepoLens v0.8.0 adds file-level impact analysis for coding agents.

## Highlights

- New `impact_analysis` MCP tool.
- Analyze outgoing dependencies from a file.
- Analyze incoming dependents that may break when the file changes.
- Suggest related files and nearby tests.
- Return a simple risk level based on incoming dependency count.

## Example MCP call

```json
{
  "name": "impact_analysis",
  "arguments": {
    "target": ".",
    "file": "src/mcp.js",
    "task": "change MCP tools",
    "limit": 20
  }
}
```

## Why it matters

After an agent chooses a file to edit, it needs to know what else to inspect. `impact_analysis` gives the agent a lightweight blast-radius view before it changes code.
