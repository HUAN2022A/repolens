# RepoLens v0.9.0

RepoLens v0.9.0 adds verification planning for coding agents through the new `test_strategy` tool.

## Highlights

- New `test_strategy` MCP tool.
- Suggests verification commands based on repository manifests.
- Finds nearby and task-relevant tests.
- Suggests files to inspect before testing.
- Reports coverage gaps when no relevant tests are detected.
- Provides concise guidance for final verification.

## Example MCP call

```json
{
  "name": "test_strategy",
  "arguments": {
    "target": ".",
    "file": "src/mcp.js",
    "task": "change MCP tools",
    "limit": 12
  }
}
```

## Why it matters

Together, `find_relevant_files`, `impact_analysis`, and `test_strategy` now give an agent a practical loop: find where to edit, understand blast radius, and choose how to verify.
